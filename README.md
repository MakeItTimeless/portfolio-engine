# Portfolio Engine

This is a personal portfolio project built with Next.js and Tailwind CSS. 
It features an interactive, simulated terminal shell in the footer for a unique 
user experience.

## Features
- Simulated command-line interface (CLI) for navigation.
- Interactive terminal with custom ASCII art.
- Security-simulated command blocks for dangerous inputs.

## Getting Started
1. Clone the repository: `git clone <your-repo-url>`
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
